# Campus-Form-Website

## A project for campus order form using Node.js , SQL , JavaScript , React-Redux 
* sever-client
* with own database using PostgreSQL
## Run: 
## client 
* npm run start
* npm install react-datepicker --save
* npm install react-select --save
* npm install emailjs-com --save
## server
* source env.sh 
* npm run start
